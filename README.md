# ofxVoiceSynthesizer (OSX)

## How to use

1. Add AppKit.framework to your project
2. Include ofxVoiceSynthesizer.h
3. Open Project Setting -> TARGETS -> Build Phases -> Compile Sources
4. Add Flag "-x objective-c++" for all file of .cpp
